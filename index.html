Explicação das Principais Tags

Tag Descrição
<html> Indica ao navegador que é um documento HTML.
<head> Armazena informações do documento, como título, CSS e metadados.
<meta charset="UTF-8"> Define a codificação UTF-8 para exibir caracteres especiais corretamente.
<title> Define o título da página, que aparece na aba do navegador e nos resultados dos motores de busca.
<body> Contém o conteúdo visível da página, como textos, imagens, links, vídeos, e tudo o que o usuário verá e interagirá.

Explicação das Tags de Formatação de Texto

h1 h2 h3 h4 h5 h6

<h1> a <h6> Cabeçalhos de diferentes niveis de importância.
Texto Formatado.

<strong>: Usada para destacar o texto, geralmente exibindo-o em negrito. É também uma tag semântica, indicando que o texto é importante.

<em>: Usada para enfatizar o texto, geralmente exibindo-o em itálico. Também possui significado semântico, sugerindo uma ênfase de significado.

<mark>: Destaca o texto, como se tivesse sido marcado com um marcador. É útil para chamar a atenção para um trecho específico de texto.

<small>: Diminui o tamanho do Texto, geralmente usado para avisos ou informações menos importantes.

<del>: Indica que o Texto foi removido ou não é mais relevante, geralmente exibindo o texto riscado.

<ins>: Indica que o Texto foi inserido, geralmente exibido como texto sublinhado.

<sub>: Exibe o texto como subscrito, útil para fórmulas químicas, como H2O.

<sup>: Exibe o texto como sobrescrito, útil para expoentes, como 23 (2 ao cubo).

Boas Práticas em HTML

As boas práticas em HTML ajudam a garantir que o código seja bem estruturado, acessível e de fácil manutenção, proporcionando uma melhor experiência tanto para desenvolvedores quanto para os usuários.

  • Indentação: Mantenha o códido bem indentado e organizado, para facilitar o entendimento.
  • Fechar todas as tags: Mesmo as tags auto-fechadas (ex:<br> para quebra de linha) devem ser usadas de forma consistente.
  • Uso de comentários: Use comentários (<!-- Comentário -->) para documentar e explicar partes importantes do código.
  • Atributos semânticos: Utilize atributos como `alt` em imagens para melhorar a acessibilidade e o SEO.
  • Organização de conteúdo: Estruture o conteúdo usando tags semânticas como <section>, <article>, <nav> para indicar áreas distintas.
  • SEO(Otimização para Motores de Busca): Use metadados, URLs amigáveis e conteúdo de qualidade para otimizar a página para motores de busca e melhorar o posicionamento.
